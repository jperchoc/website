Snake QLeanrning

This is a demo of an agent learning how to play Snake game with QLearning.

Code

Here's a sample of code

let direction = getBrainDecision(); snakeGame.snake.updateDirection(direction); let reward = getReward(); let result = snakeGame.update(); if (result === 1) { reward = brain.rewards.apple; } else if (result === -1) { reward = brain.rewards.death; } brain.rewardCount.gameReward += reward; brain.learn(reward);
